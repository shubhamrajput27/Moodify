# Moodify - Emotion-Based Music Recommendation 

An AI-powered app that detects your emotions and recommends music to match your mood.

##  Live Demo

**Frontend:** https://moodify-app.vercel.app  
**Backend API:** https://moodify-emotion-music-app.onrender.com

##  Features

- Emotion detection from text, speech, and facial expressions
- Real-time music recommendations via Spotify
- User profiles with mood history
- Mobile app support
- Dark mode theme

##  Tech Stack

**Frontend:** React, Material UI, Redux  
**Backend:** Django, Flask, MongoDB, Redis  
**AI/ML:** PyTorch, TensorFlow, BERT  

##  Quick Start

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```
