## ğŸ› Troubleshooting Guide

### Issue: Can't run the app
**Solution**: 
1. Check Node.js version (`node --version`) - need 18+
2. Install dependencies (`npm install`)
3. Check ports 3000 and 5000 are free

### Issue: Spotify API errors
**Solution**:
1. Verify Client ID and Secret in `server/.env`
2. Check credentials on Spotify Dashboard
3. Ensure app is not rate-limited

### Issue: Face detection not working
**Solution**:
1. Download models to `client/public/models/`
2. Check camera permissions in browser
3. Try HTTPS or localhost

### Issue: CORS errors
**Solution**:
1. Check `CLIENT_URL` in `server/.env`
2. Restart backend server
3. Clear browser cache

**More help**: See SETUP.md or QUICK_REFERENCE.md
